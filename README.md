# ☁️ Skycord

**Полнофункциональный клон Discord в одном HTML-файле.**
Регистрация по никнейму, кастомные профили с аватарками, создание серверов, каналы, чат в реальном времени.

---

## 🎯 Что это

Skycord — это веб-приложение, которое выглядит и работает как Discord. Открываешь ссылку — и пользуешься. Не нужно скачивать, не нужно ставить на сервер, не нужно настраивать базы данных. Один файл. Всё включено.

---

## 🧩 Возможности

### Интерфейс
| Элемент | Как в Discord |
|---|---|
| Панель серверов | Круглые иконки слева. При наведении скругляются в прямоугольник. Нажал — переключился |
| Список каналов | Категории, текстовые каналы с префиксом `#`. Активный канал подсвечен |
| Область чата | Сообщения с аватарками, именем и временем. Автоскролл вниз при новых сообщениях |
| Список участников | Правая панель. Зелёный кружок = онлайн. Ты сверху и подсвечен фиолетовым |

### Аутентификация
| Фича | Описание |
|---|---|
| Регистрация по никнейму | Без почты. Только ник и пароль |
| Хеширование пароля | SHA-256 с солью (`skycord_salt_` + никнейм) |
| Сессия | Сохраняется пока открыта вкладка. При закрытии — требуется повторный вход |
| Валидация | Никнейм минимум 2 символа (буквы/цифры/`_`/`-`). Пароль минимум 4 символа |

### Профиль
| Настройка | Как сделать |
|---|---|
| Аватар | Загрузить картинку (до 500 КБ). Автообрезка в квадрат 128×128px |
| Статус | Любой текст, отображается в профиле |
| О себе | Поле bio, до 200 символов |
| Цвет | Генерируется автоматически из никнейма (HSL) |

### Серверы
| Действие | Как работает |
|---|---|
| Создать сервер | Кнопка `+` в панели серверов → название + эмодзи-иконка |
| Присоединиться | Кнопка `🔗` → вставить ID сервера |
| Каналы | Кнопка `+` рядом с категорией → создать текстовый канал |
| Участники | Автоматически добавляются при присоединении |

### Чат
| Фича | Описание |
|---|---|
| Отправка | Enter или кнопка `➤` |
| Многострочность | Shift+Enter для переноса строки |
| Форматирование | `**жирный**`, `*курсив*`, `` `код` `` |
| Realtime | Сообщения появляются мгновенно у всех участников канала |
| История | Последние 50 сообщений подгружаются при входе |

### Участники онлайн
| Фича | Описание |
|---|---|
| Presence | Зелёный кружок рядом с теми, кто сейчас на сайте |
| Автоотключение | При закрытии вкладки/выходе — исчезает из онлайна |
| Сортировка | Ты первый в списке, остальные по алфавиту |
