# ☁️ Skycord

**Бесплатный мессенджер в стиле Discord.**  
Регистрируешься за 10 секунд, добавляешь друзей по нику, переписываешься в личных сообщениях и на серверах. Всё работает прямо в браузере.

---

## 🎯 Что тут можно делать

| Возможность | Как |
|---|---|
| **Создать аккаунт** | Придумай никнейм и пароль — и ты уже в системе |
| **Добавить друга** | Введи его ник в поле слева и нажми «Добавить» |
| **Написать в ЛС** | Кликни по другу в списке — откроется чат |
| **Создать сервер** | Нажми `+` слева, придумай название и эмодзи |
| **Зайти на сервер** | Нажми `🔍`, выбери сервер из списка и нажми «Войти» |
| **Общаться в канале** | Кликни по каналу внутри сервера — пиши |
| **Поставить аватар** | Нажми на свой никнейм внизу → «Сменить аватар» |
| **Написать статус** | Там же, в профиле |
| **Форматировать текст** | `**жирный**`, `*курсив*`, `` `код` `` |

---

## 🖼️ Как выглядит

Слева — панель с кнопками серверов и друзей.  
По центру слева — список друзей и поле для добавления.  
Справа — чат.  

Всё в тёмных тонах, как в Discord.

---

## 🚀 Как начать пользоваться

1. Открой ссылку (её даст тот, кто запустил Skycord)
2. Придумай никнейм и пароль
3. Нажми «Зарегистрироваться»
4. Добавь друга по никнейму
5. Нажми на друга в списке — пиши

---

## ❓ Частые вопросы

**«Меня не пускает, пишет ошибку»**  
Убедись, что никнейм больше 2 символов, а пароль больше 4. Никнейм можно писать русскими или английскими буквами.

**«Как удалить друга?»**  
В списке друзей нажми `✕` рядом с именем.

**«Как выйти из сервера?»**  
Нажми на иконку ☁️ (главная) слева — вернёшься к списку друзей.

**«Я забыл пароль»**  
Пароль восстановить нельзя. Создай новый аккаунт.

**«Кто угодно может зайти под моим ником?»**  
Нет. Пароль хешируется, никто не узнает твой пароль. Но никнейм виден всем — как в любой соцсети.

**«Можно ли писать на телефоне?»**  
Да, интерфейс подстраивается под экран. Но удобнее на компьютере.
